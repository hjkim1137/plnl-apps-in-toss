# 주간 목표 도장판 수동 테스트 시나리오

> 테스트 환경: `npm run dev` 실행 후 로컬호스트 접속 → **월간현황** 탭

---

## T-1. 도장 개수 확인 (주차 수)

**목적**: 달에 따라 도장이 4개 또는 5개 나오는지 확인

| 이동할 달 | 예상 도장 수 |
|---|---|
| 2026년 2월 | 4개 |
| 2026년 6월 | 5개 |
| 2026년 3월 | 5개 |

**방법**
1. 월간현황 탭 → 달력 상단 `‹ ›` 로 위 달로 이동
2. 달력 하단 도장판의 원 개수 세기

**확인 포인트**
- [ ] 도장 수 = 달력 행 수와 일치
- [ ] 각 원 아래 `1주`, `2주`, … 라벨이 순서대로 표시됨
- [ ] 헤더에 `(N회/주)` 가 표시됨

---

## T-2. 주간 목표 횟수 표시 확인

**목적**: 월간 목표에서 주간 목표가 올바르게 도출되는지 확인

**방법**
1. 오늘 탭 → 목표 횟수를 **12회**로 설정
2. 월간현황 탭 이동 → 도장판 헤더의 `(N회/주)` 확인

**예상값**

| 월간 목표 | 이번 달 주차 수 | 예상 주간 목표 |
|---|---|---|
| 12회 | 5주 달 | 3회/주 |
| 12회 | 4주 달 | 3회/주 |
| 8회 | 4주 달 | 2회/주 |
| 9회 | 4주 달 | 3회/주 |
| 1회 | 4주 달 | 1회/주 |

- [ ] 헤더 `(N회/주)` 숫자가 위 표와 일치함

---

## T-3. 미달성 상태 표시

**목적**: 주간 목표를 채우지 못한 주차의 빈 도장 표시 확인

**방법**
1. 이번 달 기록이 비어 있는 상태에서 월간현황 탭 확인
2. 달력에서 1주차에 목표 횟수 **미만**으로만 운동함 기록

**확인 포인트**
- [ ] 달성 전 원: 회색 배경 + 점선 테두리
- [ ] 원 안에 `현재횟수/목표횟수` (예: `1/3`) 숫자 표시
- [ ] 녹색 도장·👍 이모지가 보이지 않음

---

## T-4. 달성 시 도장 + 애니메이션

**목적**: 주간 목표 달성 순간 도장이 찍히고 애니메이션이 재생되는지 확인

**방법**
1. 월간 목표를 **4회**로 설정 (주간 목표가 낮아 빠르게 테스트 가능)
2. 이번 달 1주차에 운동함 기록을 하나씩 늘려가며 목표 횟수 도달 순간 확인
   - 달력 날짜 탭 → done/missed 토글

**확인 포인트**
- [ ] 목표 횟수 달성 순간 도장이 **팝 + 회전** 애니메이션과 함께 등장
- [ ] 도장: 녹색 그라데이션 원 + 👍 이모지
- [ ] 그림자(초록빛 glow) 표시됨

---

## T-5. 달성 취소 (도장 사라짐)

**목적**: 달성 후 기록을 취소하면 도장이 빠지는지 확인

**방법**
1. T-4 달성 상태에서 해당 주차 날짜 하나를 다시 탭 → missed 또는 빈칸으로 변경
2. 달성 횟수가 주간 목표 미만으로 떨어지는 순간 확인

**확인 포인트**
- [ ] 녹색 도장이 사라지고 회색 빈 원으로 돌아옴
- [ ] 원 안에 `현재횟수/목표횟수` 숫자 다시 표시
- [ ] 다시 목표 채우면 애니메이션 재생됨 (애니메이션 재실행 확인)

---

## T-6. 과거 달 조회

**목적**: 과거 달 도장판이 그 달 기록 기준으로 표시되는지 확인

**방법**
1. `‹` 버튼으로 지난달로 이동
2. 도장판 확인

**확인 포인트**
- [ ] 지난달 기록이 있으면 달성 주차에 도장 표시
- [ ] 기록이 없는 달은 모든 도장이 `0/N` 빈 상태
- [ ] 현재 달 목표를 바꿔도 과거 달 도장판은 **변하지 않음** (스냅샷 기준)

---

## T-7. 화면 배치 확인

**목적**: 도장판 위치와 구분선이 올바른지 확인

**확인 포인트**
- [ ] 달력 그리드 → 운동함/운동안함 **범례** → **구분선** → 도장판 순서
- [ ] 구분선이 범례와 도장판 헤더 사이에 표시됨
- [ ] 도장판 헤더: `주간 목표 달성 (N회/주)`
- [ ] 도장들이 가로로 중앙 정렬됨
- [ ] 5주 달에서 도장 5개가 화면 밖으로 잘리지 않음

---

## T-8. 엣지 케이스

### T-8a. 첫 주에 날이 1일뿐인 달 ✅ QA K-9 커버됨
- **토요일 시작 달 예시: 2026-08 (8월)** → lead=6, 첫 행에 8/1 하루만 존재
  - (참고: 2026-03(3월)은 일요일 시작(lead=0)이라 첫 행이 1~7일로 꽉 차므로 해당 없음)
- **방법**: 달력 상단 `‹ ›`로 2026년 8월로 이동
- **확인**: 1주차 원 안에 `0/N` 또는 `1/N` 표시되고 도장 안 찍힘
  - (8/1 하루만 있어서 weeklyGoal이 3 이상이면 달성 자체가 불가)

### T-8b. 마지막 주에 날이 1~3일뿐인 달 ✅ 코드 정상 처리 확인
- **예시**: 31일 달에서 lead=0이면 마지막 행이 29~31일 (3일만 존재)
- **방법**: 마지막 주차 날짜 전부를 done으로 채우기
- **확인**: done 수 ≥ weeklyGoal이면 마지막 주 도장 찍힘
  - 부분 주차도 동일한 weeklyGoal 기준으로 판정함 (비례 처리 없음)

### T-8c. 월간 목표 변경 직후 ✅ 상태 연동 확인
- **주의**: 목표 변경 UI는 **오늘 탭**에 있음 (월간현황 탭에는 설정 UI 없음)
1. 오늘 탭 → 목표 횟수 변경
2. 월간현황 탭으로 이동
- **확인**: 헤더 `(N회/주)` 숫자가 변경된 목표 기준으로 즉시 갱신됨, 도장 달성 여부도 재계산됨
