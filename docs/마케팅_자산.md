## 1. 앱 소개 문구

### 짧은 설명 (40자 이내 — 앱인토스 리스트 노출)
```
운동비, 얼마나 뽑았어요? 출석 체크 하나로 회수율 확인
```

### 한 줄 슬로건
```
운동비 회수율 0%는 호구의 증거 💳
```

### 중간 설명 (80~100자 — 앱 소개 상단)
```
결제만 하면 기부, 가면 회수. 
오늘 운동하러 갔는지만 체크하면 
이번 달 운동비를 얼마나 돌려받았는지 바로 확인할 수 있어요.
```

### 긴 설명 (앱인토스 상세 설명란)
```
헬스장 결제하고 안 가면 돈은 그냥 기부되는 거 알죠?

뺄래 낼래는 그 기부 금액을 숫자로 보여줘요.
출석 체크 하나만 하면 됩니다.

📊 오늘 갔어요 → 단가만큼 회수
📉 오늘 안 갔어요 → 단가만큼 기부

이달 회수율이 높을수록 당신은 호구가 아닙니다.
100% 달성하면 본전 졸업장 발급해드려요 🎓

— 기능 요약 —
• 이번 달 회수율·기부액 실시간 계산
• 7단계 구간 피드백 ("지갑만 운동러" → "이젠 내가 갑")
• 연속 출석 스트릭 & 등급 시스템
• 월말 결산 리포트 + 표창장 공유
• 비로그인도 핵심 기능 이용 가능
```
